## Списки

Чтобы добавить ненумированные списки, необходимы пункты выделить звёздочкой (*). Например, вот так;
* Элемент один.
* Элемент два.
* Элемент три.

Чтобы добавить нумерованные списки, необходимо пункты просто пронумировать. Например, вот так;
1. Элемент первый.
2. Элемент второй.
